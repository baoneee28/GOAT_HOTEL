# AGENTS.md

> **Dành cho tất cả AI coding agents**: Đọc `.antigravity/rules.md` trước khi thực hiện bất kỳ thao tác nào trong repo này.

## Vị trí Rules Chính Thức

Toàn bộ quy tắc làm việc cho repo này được lưu tại:

```
.antigravity/rules.md
```

File này chỉ là cầu nối định hướng. Nó không chứa quy tắc nào trong chính nó.

## Tóm Tắt Nhanh

| Thuộc tính | Giá trị |
|------------|---------|
| Dự án | GOAT Hotel — Ứng dụng quản lý khách sạn |
| Backend | Spring Boot 3.2 · Java 17 · Maven · JPA · SQL Server |
| Frontend | React 19 · Vite · React Router v7 · Axios |
| Trạng thái | **Chưa hoàn thành** — đang phát triển tích cực, đang refactor |
| Nguồn rules | `.antigravity/rules.md` |

## Bắt Buộc Không Được Vi Phạm

- **Không được** chỉnh sửa `backend/target/`, `frontend/node_modules/`, `frontend/dist/`, hoặc `backend/apache-maven-*/`.
- **Không được** coi các file `backend/*.txt`, `backend/*.log`, `backend/err*` là nguồn thông tin đáng tin cậy.
- **Không được** triển khai hoặc đoán mò các tính năng chưa được xác nhận.
- Khi không chắc chắn, hãy thực hiện **thay đổi nhỏ nhất, an toàn nhất, dựa trên bằng chứng** có thể.
